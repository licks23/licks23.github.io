# 工业遗产改造与保护H5互动广告

## 项目介绍

本项目是为第四届工业遗产改造与保护创新设计大赛创作的H5互动广告作品。作品旨在宣传工业遗产的保护与改造，探索老工业城市转型发展新路径。

## 技术栈

- HTML5
- CSS3
- JavaScript

## 项目结构

```
.
├── index.html          # 主页面文件
├── style.css           # 样式文件
├── script.js           # 交互脚本
└── README.md           # 项目说明
```

## 作品特点

1. **响应式设计**：适配手机屏幕尺寸，默认页面宽度640px，高度1008px
2. **互动体验**：包含页面切换、选项选择、滑动手势等互动功能
3. **视觉效果**：采用现代化的设计风格，丰富的动画效果
4. **内容丰富**：涵盖工业遗产介绍、辽宁工业遗产、成功改造案例、保护意义等内容
5. **互动环节**：设置了互动投票环节，增强用户参与感

## 如何使用

### 本地运行

1. 确保安装了Python
2. 在项目目录下启动本地服务器：
   ```
   python -m http.server 8000
   ```
3. 在浏览器中访问：http://localhost:8000

### 访问方式

- 直接在浏览器中打开index.html文件
- 部署到Web服务器上，通过URL访问

## 作品内容

1. **封面页**：展示主题和副标题
2. **工业遗产介绍**：解释工业遗产的定义和范围
3. **辽宁工业遗产**：介绍辽宁的重要工业遗产资源
4. **成功改造案例**：展示国内外成功的工业遗产改造案例
5. **保护的意义**：阐述保护工业遗产的重要性
6. **互动环节**：用户投票选择工业遗产改造方式
7. **结语**：总结主题，呼吁共同保护工业遗产

## 交互说明

- **按钮点击**：点击"上一页"、"下一页"按钮切换页面
- **滑动手势**：左右滑动屏幕切换页面
- **页码指示器**：点击底部的圆点直接跳转到对应页面
- **选项选择**：在互动环节点击选项进行选择
